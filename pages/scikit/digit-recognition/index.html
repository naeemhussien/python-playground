Digit Recognition Using Support Vector Machines

This is an application that demonstrates digit recognition of hand written digits. This calls upon the following code hosted on PythonAnywhere via webservice:

  • REST web service using Flask.
  • Support Vector Machine algorithm to perform classification using SciKit library.
